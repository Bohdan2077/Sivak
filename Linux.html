Основи GNU/Linux

Коротка інформація

GNU/Linux — це операційна система, що поєднує в собі ядро Linux і програмне забезпечення, розроблене в рамках проекту GNU. Основні характеристики GNU/Linux:

  • Відкритий код: GNU/Linux є системою з відкритим кодом, що означає, що її вихідний код доступний для всіх. Користувачі можуть вивчати, змінювати та поширювати програмне забезпечення.
  • Портативність: Операційна система може працювати на різних типах апаратного забезпечення, включаючи ПК, сервери, вбудовані системи та мобільні пристрої.
  • Мультизадачність: GNU/Linux підтримує виконання кількох процесів одночасно, що дозволяє користувачам запускати декілька програм в один і той же час.
  • Безпека: Система має вбудовані функції безпеки, такі як контроль доступу, що дозволяє зменшити ризики зловживання.
  • Дистрибутиви: Існує безліч дистрибутивів GNU/Linux (наприклад, Ubuntu, Fedora, Debian), кожен з яких має свої унікальні особливості та орієнтацію на різні користувацькі потреби.
  • Командний рядок та графічні інтерфейси: GNU/Linux надає потужні можливості командного рядка для адміністраторів, а також різні графічні інтерфейси для кінцевих користувачів.

Ця операційна система широко використовується в серверному середовищі, розробці програмного забезпечення, наукових дослідженнях та багатьох інших сферах.

Команди Linux CLI

  • pwd — показати поточний каталог
  • echo — вивести текст
  • ls — список файлів у каталозі
  • cd — змінити каталог
  • touch — створити порожній файл
  • mkdir — створити нову директорію
  • cp — копіювати файли
  • mv — перемістити файли
  • rm — видалити файли
  • history — показати історію команд

BASH-скрипт авторизації

                
          #!/usr/bin/env bash
          # Простий скрипт авторизації
          
          correct_username="user"
          correct_password="password"
          
          echo "Введіть ім'я користувача:"
          read username
          
          echo "Введіть пароль:"
          read -s password
          
          if [ "$username" == "$correct_username" ] && [ "$password" == "$correct_password" ]; then
              echo "Авторизація успішна. Ласкаво просимо!"
          else
              echo "Невірне ім'я користувача або пароль."
          fi
